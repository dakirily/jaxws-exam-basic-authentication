Example shows different possibilities of basic authentication scenarios in JAX-WS:

Class com.tsi.jaxws.exam.service.CustomerService01 shows basic authentication using SOAPHandler.
Class com.tsi.jaxws.exam.service.CustomerService02 shows basic authentication in web service method body.
Class com.tsi.jaxws.exam.service.CustomerService03 shows basic authentication using container security.

Client is implemented in unit test: com.tsi.jaxws.exam.WsTest
